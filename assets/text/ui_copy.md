# UI Copy - Last Light Courier (MVP)

## Buttons
- Start Run
- Restart Run
- Resume
- Pause
- Quit to Title

## Tooltips
- Move: "WASD or Arrow Keys"
- Dash: "Space: burst move, then cooldown"
- Cells: "Collect 6 cells to unlock extraction"
- Multiplier: "Grows every 10s without damage"
- Extraction: "Opens after cell quota is met"

## HUD / Alerts
- Start: "Run live. Build a clean route."
- Mid warning (121s): "Grid surge. Patrol traffic increasing."
- Final warning (<30s): "Thirty seconds. Commit to extraction now."
- Damage: "Integrity hit. Multiplier reset."
- Extraction unlocked: "Quota met. Extraction is now active."

## End Screen
- Win title: "Extraction Secured"
- Win body: "Delivery complete. Last light preserved."
- Fail title: "Signal Lost"
- Fail body: "Mission failed. Re-run and reroute."

## Error Messages
- Save failure: "Best score not saved. Retry."
- Invalid action: "Action blocked while paused."
